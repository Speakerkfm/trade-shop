## Trade-shop

## Installation
1. Download project
```bash
$ cd ~/go/src
$ git clone https://github.com/Speakerkfm/trade-shop
```
2. Create database
```bash
$ make migrations
```

3. Configure .env
```bash
$ cp .env.dist .env
```

4. Run tests
```bash
$ make test
```

5. Run project
```bash
$ make run
```

## Documentation
```bash
$ make doc
```
